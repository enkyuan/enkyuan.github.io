## Rundown

So think of Foursquare mixed with volunteering tailored to high schoolers. That's basically what this app is for. It was my first major personal project, so nothing too crazy in terms of the tech stack--Flutter, Firebase, and some middleware from Google Cloud to add some flair. Check out the repo (below). The project is currently archived and it's turned into a can of worms (lots of old packages and bugs since I stopped maintaining it last year).
