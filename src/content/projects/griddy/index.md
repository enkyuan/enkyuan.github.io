## Overview

I read some high-level stuff on banking systems and the tech they use to work (none of the nitty-gritty details of course). This project is a guinea pig for my intro to building APIs and designing servers to work with databases and the like. The banking functionality was just a sidequest and me trying to find something to apply some technologies too. It does some very basic stuff like creating accounts, processing transactions, etc. Nothing too excessive. Written in Go. I used a Postgres image on docker to store all the data, JWT Auth to secure the requests, and Postman to test this code. Check out what I wrote in the repo.
